# RESTful Stock API End Points

# GET /api/stocks (List of Stocks)
# GET /api/stocks/stockId  (Get Stock from the list by its ID)
# PUT /api/stocks/stockId (Update current_price/name of a single Stock) 
# DELETE /api/stocks/stockId  (Delete a single Stock by its ID) 
# POST /api/stocks (Create a new Stock)
